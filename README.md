# TORTUETOPIA – Thème WordPress

Thème WordPress personnalisé réalisé dans le cadre d’un projet étudiant.

## Description
Ce thème a été conçu pour le site de l’association TORTUETOPIA, dédiée à la protection et à l’adoption responsable des tortues.

## Fonctionnalités
- Design responsive (mobile & desktop)
- Personnalisation CSS selon la charte graphique
- Structure basée sur l’arborescence du projet
- Compatibilité avec l’éditeur de blocs WordPress
